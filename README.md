# hanja-battle-game

# Purpose
This project is a Unity-based educational battle game that uses Hanja learning as the core battle mechanic. The goal is to create a turn-based prototype where players learn, review, and use Hanja characters through combat, progression, and memory reinforcement.

The project is built around a structured database of at least 1,000 Hanja characters. Each character is intended to have a reading, meaning, difficulty tier, type, and gameplay use, so the learning system can grow without manually hardcoding every character into the game.
