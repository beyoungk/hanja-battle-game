# hanja-battle-game

# Purpose
This project is a Game Maker-based educational battle game that uses Hanja learning as the core battle mechanic. The goal is to create a prototype where players learn, review, and use Hanja characters through combat and memory reinforcement, with the entertainment of collection of characters into a roster.

The project is built around a structured database of 1,000 Hanja characters. Each character will have a reading, meaning, difficulty tier, and combat class.
